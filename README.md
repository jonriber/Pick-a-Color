# Pick-a-Color
Pick a Color Game, to Practice React.js with Ts <br/>
Practice of React interview challenges.

### Install & Usage:
Enter directory: <br/>
```
  cd Pick-a-color
```

Install dependencies: <br/>
```
  npm install
```

Run development mode:<br/>
```
  npm run dev
```
